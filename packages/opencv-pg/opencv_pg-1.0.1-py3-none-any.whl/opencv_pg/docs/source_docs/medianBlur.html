Blurs an image using the median filter.

{% endblock %} {% block signature %} 
cv2.medianBlur(src, ksize[, dst]) → dst
{% endblock %} {% block parameters %}
  • src (np.ndarray): Input 1-, 3-, or 4-channel image; when ksize is 3 or 5, the image depth should be cv2.CV_8U, cv2.CV_16U, or cv2.CV_32F, for larger aperture sizes, it can only be cv2.CV_8U.
  • ksize (int): Aperture linear size; it must be odd and greater than 1, for example: 3, 5, 7, etc. The ksize indicates the size of the square kernel to be used by this filter, i.e. how large of a neighborhood to use surrounding each pixel.
  • dst (optional; np.ndarray): Destination array of the same size and type as src.
{% endblock %} {% block explanation %}

The function smoothes an image using the median filter with the ksize x ksize aperture. Each channel of a multi-channel image is processed independently. In-place operation is supported.

{% endblock %} {% block notes %}
  • The median filter runs through each element of the signal (in this case the image) and replaces each pixel with the median of its neighboring pixels (located in a square neighborhood around the evaluated pixel).
  • In median blurring, the central element is always replaced by some pixel value in the image.
  • This is highly effective against salt-and-pepper noise in an image.
  • The median filter uses BORDER_REPLICATE internally to cope with border pixels.
